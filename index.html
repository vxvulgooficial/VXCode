Tudo que você precisa

VXCode foi construído com Monaco Editor — o mesmo motor do VSCode. Rápido, inteligente, bonito.

⚡
Monaco Editor
O mesmo motor do VSCode. Autocomplete, syntax highlight, folding, minimap e muito mais.
🌙
Suporte a Lua
Suporte nativo a Lua com syntax highlight personalizado. Perfeito para MTA:SA e outros projetos.
🔍
Busca Inteligente
Busca em todos os arquivos com navegação direta para a linha exata. Suporte a RegEx.
💻
Terminal Integrado
Terminal real integrado no editor. CMD no Windows, Bash no Linux.
💾
Sessão Automática
Lembra tudo que você deixou aberto. Fecha e abre — está tudo lá.
🖼️
Visualizador de Imagens
Abre PNG, JPG, GIF, SVG direto no editor. Sem precisar de outro programa.
🎨
Temas Personalizáveis
7 temas incluídos. Cores totalmente customizáveis. Faz o editor do seu jeito.
🎮
Discord Rich Presence
Mostra o que você está editando no Discord. A rapaziada vai ver que você tá no VXCode.
🔴🟡🟢
Bolinhas macOS
Titlebar elegante com bolinhas coloridas estilo macOS. Bonito demais.
